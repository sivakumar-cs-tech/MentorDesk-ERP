 🚀 MentorDesk ERP

A Full Stack Training Institute Management System built using the MERN Stack.

---

 📌 Overview

MentorDesk ERP is a modern Training Institute Management System developed to manage students, courses, attendance, tasks, timelines and reports efficiently.

The application provides a professional dashboard with analytics and helps training institutes track student progress and daily activities in a centralized system.
